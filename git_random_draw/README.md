# 学生随机抽奖系统

基于 Java Swing 开发的本地桌面应用，全部数据通过 `data` 目录下的 TXT 文件读取和保存。

## 功能

- 管理：读取 `student_information.txt`，支持新增、编辑、单条或批量删除、按班级筛选
- 抽奖：支持考勤抽奖和回答问题抽奖，多班级参与、随机无重复抽取、结果标记并保存
- 统计：读取考勤/表现 TXT 数据，自绘柱状图并联动展示记录列表

## 运行

```powershell
mkdir out -ErrorAction SilentlyContinue
javac -encoding UTF-8 -d out (Get-ChildItem -Recurse -Filter *.java | ForEach-Object FullName)
java -cp out com.randomdraw.Main
```


提示词：
Java学生抽奖系统开发需求提示词
 
需求：使用Java开发一款学生随机抽奖系统，全程通过读取本地TXT文件获取、存储数据，界面设计要求高级有质感，杜绝庸俗简陋的视觉效果；学生信息录入规则为：学号、班级为必填项，姓名为非必填项。
 
一、系统依赖TXT数据文件（格式规范）
 
1. student_information.txt：首行固定为「学号 姓名 班级」，存储学生基础信息

2. student_attendance.txt：首行固定为「学号 姓名 班级 考勤情况」，存储学生考勤数据

3. student_performance.txt：首行固定为「学号 姓名 班级 表现情况」，存储学生表现数据
 
二、系统核心界面与功能需求
 
系统主界面固定展示三大功能入口：管理、抽奖、统计，点击对应入口跳转至对应功能页面，各模块需求如下：
 
1. 管理界面
 
- 基础数据读取：读取student_information.txt文件中的学生信息，以清晰版式展示学号、姓名、班级全部字段

- 信息管理功能：支持学生信息编辑、单条/批量删除，支持新增学生信息（新增时校验学号、班级必填，姓名可选填）

- 筛选功能：支持按班级精准筛选，仅展示对应班级的学生信息
 
2. 抽奖界面
 
- 抽奖类型选择：提供「考勤抽奖/回答问题抽奖」两种模式供用户选择

- 抽奖条件设置：支持多选多个班级参与抽奖，用户可自定义填写抽奖人数

- 抽奖展示效果：根据选定班级、抽奖人数，从对应学生数据中随机无重复抽取，结果展示界面需做高级视觉设计，提升交互质感

- 数据标记与存储：

- 考勤抽奖模式：支持为抽中的学生标记「出勤/缺勤」考勤状态，标记后自动将数据保存至student_attendance.txt

- 回答问题抽奖模式：支持为抽中的学生标记「优秀/良好/差」表现等级，标记后自动将数据保存至student_performance.txt
 
3. 统计界面
 
需先选择统计类型：「考勤情况统计/表现情况统计」，两类统计页面需求如下：
 
- 考勤情况统计界面：

1. 读取student_attendance.txt全部数据

2. 生成直观柱状图，展示所有学生的出勤次数、缺勤次数统计

3. 列表展示全部考勤记录，支持按班级筛选数据，筛选后图表与列表同步更新

- 表现情况统计界面：

1. 读取student_performance.txt全部数据

2. 生成直观柱状图，展示所有学生的优秀、良好、差表现情况统计

3. 列表展示全部表现记录，支持按班级筛选数据，筛选后图表与列表同步更新
